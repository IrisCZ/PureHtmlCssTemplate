Buttons

Default Buttons
Use <button> and add class btn and btn-info, btn-success, btn-warning or btn-error.
e.g. <button> class="btn btn-info"
Outlined Buttons
As default buttons adding class outlined.
e.g. <button> class="btn-info outlined"
Gradient Buttons
Add classes btn-grad-info,... and gradient.
e.g. <button> class="btn btn-grad-info gradient"
Clear Buttons
Use classes info, success, warming or error and clear.
e.g. <button> class="clear info"

Grid

12 columns grid with offset
Use always a <div class="grid-container"> Every cell must have width and height. Use offset to position cells in a grid.
e.g. <div> class="col-xs-1 col-xs-offset-12"
12 columns grid with cells of multiple sizes
Use xs, sm, md or lg to adapt the grid to diferent sizes of screens
e.g. <div class="col-xs-12 col-md-6"> Means that your cell will have full with in a very small screen and half with in a mediun screen.

Switches

Default Switches
Use <label> and add class switch and bg-info, bg-success, bg-warning or bg-error.
e.g. <label> class="switch-label bg-info"
Linear Switches
As default switches adding class linear e.g. <label> class="switch-label bg-info linear"
Rounded Switches
As default switches adding class rounded e.g. <label> class="switch-label rounded"
